# Carpeta de Imágenes para la Página de Niyati

## Instrucciones

Coloca las siguientes fotos en esta carpeta:

### Imagen Principal del Hero (Pantalla de inicio)
- `hero-niyati.jpeg` - La imagen de fondo principal que aparece en la primera pantalla

### Galería de Fotos
1. `niyati-1.jpeg` - Primera foto de la galería
2. `niyati-2.jpeg` - Segunda foto de la galería
3. `niyati-3.jpeg` - Tercera foto de la galería
4. `niyati-4.jpeg` - Cuarta foto de la galería

**Formatos soportados:** .jpg, .jpeg, .png, .webp

**Recomendaciones:**
- Para la imagen hero (hero-niyati.jpg): usa una foto horizontal de alta resolución (mínimo 1920x1080px)
- Para la galería: cualquier tamaño y orientación funciona

Las imágenes ya están configuradas para cargar automáticamente desde esta carpeta.
